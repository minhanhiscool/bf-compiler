## Usage
### Installation
Compile the code yourself or get the [release binary](https://github.com/minhanhiscool/bf-compiler/releases).
Either way, you will need to have:
- a Linux system;
- [nasm](https://www.nasm.us/);
- a linker, preferably [ld](https://gcc.gnu.org/);

### Running:
```
rabtac [options] <input file>
```

Your output will be written to ./out.asm, or ./out.o, or ./out, depending on your options.
```
```

### Arguments:
```
-a: output assembly code only
-c: output object files only
-o: optimize output

-a and -c are mutually exclusive
```
